# Live Write Ups

## 🎯 Goal
Turn static security writeups into interactive, branching decision-making journeys (similar to [[hackthrough.live]]). Users will be presented with scenarios and must choose the correct path, receiving feedback on their choices to improve their real-world bug-hunting intuition.

## 🗺️ Milestones
- [ ] 1. Define JSON/MDX Schema for branching scenarios.
- [ ] 2. Design Astro JS UI/UX (Decision cards, Feedback loops).
- [ ] 3. Create `AGENTS.md` and `GEMINI.md` for project-specific automation.
- [ ] 4. Implement "Writeup Importer" skill (CLI-based).
- [ ] 5. Set up GitHub repository and CI/CD for static site hosting.
